# Rôle des principaux composants Sylius

## Objectif
Savoir à quoi sert chaque brique.

## Documentation Officielle
*   [Components Documentation](https://docs.sylius.com/en/latest/components_and_bundles/components/index.html)

*   **Addressing** : Pays, Provinces, Zones.
*   **Attribute** : Attributs dynamiques (EAV) pour produits, etc.
*   **Channel** : Gestion multicanal.
*   **Currency** : Devises et taux.
*   **Customer** : Profil client (séparé du User pour l'auth).
*   **Inventory** : Stock physique.
*   **Locale** : Langues.
*   **Order** : Panier et Commande.
*   **Payment** : Paiements.
*   **Product** : Catalogue (Produits, Variantes, Associations).
*   **Promotion** : Règles de réduction.
*   **Resource** : CRUD générique (la fondation de tout).
*   **Review** : Avis clients.
*   **Shipping** : Livraison.
*   **Taxation** : Taxes.
*   **User** : Authentification (Admin / ShopUser).
*   **Mailer** : Envoi d'emails.
*   **Grid** : Gestion des tableaux de données en admin.
