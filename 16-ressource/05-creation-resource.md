# Création d'une Ressource pas à pas

1.  Créer l'Entité Doctrine (`src/Entity/MailingList.php`).
2.  Déclarer la ressource dans `sylius_resource.yaml`.
3.  (Optionnel) Créer le FormType.
4.  (Optionnel) Définir les routes dans `config/routes/app_mailing_list.yaml` type `sylius_resource`.
5.  (Optionnel) Définir la Grille dans `sylius_grid.yaml`.

Vous avez un CRUD complet opérationnel en < 10 minutes.
