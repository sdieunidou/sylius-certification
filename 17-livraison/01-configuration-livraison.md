# Configuration des méthodes de livraison

## Objectif
Créer des options d'expédition.

## Modèle
*   **ShippingMethod** : L'option visible (ex: "DHL Express").
*   **Zone** : Où est-ce disponible.
*   **Category** : Pour quel type de produit (ex: "Produits Frais").
*   **Calculator** : Comment on calcule le prix.

## Admin
**Configuration > Shipping Methods**.
L'interface change dynamiquement les champs de configuration selon le calculateur choisi (ex: poids par palier vs prix fixe).

## Traduction
Le nom et la description sont traduisibles pour s'afficher dans la langue du client au checkout.
